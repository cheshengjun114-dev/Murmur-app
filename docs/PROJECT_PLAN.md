# Murmur 프로젝트 기획서

## 1. 제품 개요

### 제품 이름

Murmur

### 한 줄 소개

20~30대가 익명으로 고민, 일상, 썰을 공유하고 공감할 수 있는 커뮤니티 플랫폼

### 만들게 된 이유

기존 SNS는 팔로워, 좋아요 수 등 비교와 과시 중심 문화가 강해 부담을 느끼는 경우가 많습니다.
그래서 사용자가 익명으로 편하게 글을 작성하고, 공감과 댓글을 통해 소통할 수 있는 커뮤니티 서비스를 만들고자 했습니다.

또한 단순 게시판이 아니라 신고, 블라인드 처리, 인기글 기능 등을 포함해 실제 서비스 운영 흐름까지 고려한 프로젝트를 목표로 합니다.

## 2. 타겟 사용자

### 주요 사용자

- 20~30대 대학생
- 취업 준비생
- 직장인
- 익명으로 일상이나 고민을 공유하고 싶은 사용자

### 사용자가 원하는 것

- 부담 없이 익명으로 글을 작성하고 싶다.
- 회사, 취업, 연애, 인간관계 고민을 편하게 공유하고 싶다.
- 다른 사람들의 공감이나 댓글 반응을 받고 싶다.
- 인기글이나 썰을 보며 재미를 느끼고 싶다.
- 기존 SNS보다 비교와 과시가 적은 커뮤니티를 원한다.

## 3. 핵심 기능

### 높음

- 회원가입 / 로그인: JWT 기반 사용자 인증
- 게시글 CRUD: 익명 게시글 작성, 수정, 삭제, 조회
- 댓글 / 대댓글: 게시글에 댓글 및 답글 작성
- 반응 시스템: 공감해요, ㅋㅋㅋ, 힘내요 반응 기능
- 신고 / 블라인드: 신고 누적 시 게시글 자동 숨김 처리

### 중간

- 인기글 기능: 조회수, 댓글수, 반응수를 기반으로 인기글 제공
- 익명 번호 시스템: 게시글 단위 익명1, 익명2 형태로 표시
- 카테고리 검색: 회사, 연애, 고민 등 카테고리별 조회

### 낮음

- 북마크 기능: 게시글 저장 기능
- 관리자 페이지: 신고 게시글 관리 기능

## 4. 화면 목록

- 로그인 화면
- 회원가입 화면
- 메인 홈 화면: 최신글 / 인기글
- 게시글 상세 화면
- 게시글 작성 화면
- 게시글 수정 화면
- 카테고리별 게시글 화면
- 마이페이지
- 북마크 목록 화면
- 관리자 신고 관리 화면

## 5. 메인 화면 흐름

```text
로그인
-> 메인 홈 화면
-> 게시글 조회
-> 게시글 상세
-> 댓글 / 반응 작성
-> 게시글 작성
-> 인기글 조회
-> 마이페이지 이동
```
