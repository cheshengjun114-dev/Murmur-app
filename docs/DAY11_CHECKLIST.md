## API

### 북마크 저장/해제

```http
POST /api/posts/{postId}/bookmark
Authorization: Bearer {accessToken}
```

응답:

```json
{
  "success": true,
  "message": "요청이 성공했습니다.",
  "data": {
    "postId": 1,
    "bookmarked": true,
    "bookmarkCount": 1
  }
}
```

### 내 북마크 목록

```http
GET /api/me/bookmarks?page=0&size=10
Authorization: Bearer {accessToken}
```

## 정책

- 로그인한 사용자만 북마크할 수 있습니다.
- 같은 사용자는 같은 게시글을 한 번만 북마크할 수 있습니다.
- 이미 북마크한 게시글에서 저장 버튼을 다시 누르면 북마크가 해제됩니다.
- 삭제된 게시글은 내 북마크 목록에 표시하지 않습니다.
- 게시글 상세 조회 시 현재 로그인 사용자의 북마크 여부를 함께 내려줍니다.
