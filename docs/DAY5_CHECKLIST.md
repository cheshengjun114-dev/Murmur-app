## 프론트엔드 API 연결

프론트엔드는 아래 기본 주소로 백엔드 API를 호출합니다.

```text
VITE_API_BASE_URL=http://localhost:8080/api
```

## 확인할 화면

```text
/
/categories/company
/posts/new
/posts/{postId}
/posts/{postId}/edit
```

## 주의점

- 백엔드가 먼저 실행되어 있어야 합니다.
- RDS 프로필을 쓰면 IntelliJ Active profiles에 `rds`가 있어야 합니다.
- 댓글과 반응 기능은 6~7일차에 구현하므로 현재는 `0`으로 표시됩니다.
