## 인기글 점수 정책

```text
인기점수 = 조회수 + 댓글수 * 3 + 반응수 * 2
```

- 조회수는 관심도 지표입니다.
- 댓글은 사용자가 시간을 들여 참여한 행동이므로 3점을 줍니다.
- 반응은 가벼운 참여 행동이므로 2점을 줍니다.
- 신고로 숨김 처리된 게시글은 인기글에 노출하지 않습니다.

## 인기글 API

### 전체 인기글

```http
GET /api/posts/popular?page=0&size=5
```

### 카테고리별 인기글

```http
GET /api/posts/popular?categoryId={categoryId}&page=0&size=5
```

## 화면 반영

- 홈 화면
  - 인기글
  - 최신글
- 카테고리 화면
  - 해당 카테고리 인기글
  - 해당 카테고리 최신글
