# Murmur Frontend

React 기반 Murmur 프론트엔드 프로젝트입니다.

## 예정 기술 스택

- React
- Vite
- Axios
- React Query
- TailwindCSS

## 폴더 구조

```text
src/
  app/          # 라우터, 전역 Provider
  assets/       # 이미지, 아이콘 등 정적 리소스
  components/   # 공통 UI 컴포넌트
  features/     # 기능 단위 모듈
  hooks/        # 공통 커스텀 훅
  lib/          # axios, react-query 등 라이브러리 설정
  pages/        # 페이지 컴포넌트
  styles/       # 전역 스타일
  types/        # 공통 타입
```
